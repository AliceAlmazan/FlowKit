# FlowKit
Intuitive Python framework for flow cytometry analysis and visualization, including GatingML support.

## Installation

### From PyPI

`pip install flowkit`

### From source

```
git clone https://github.com/whitews/flowkit
cd flowkit
python setup.py install
```

## Requirements

* flowio
* flowutils
* matplotlib
* scipy
* seaborn
* pandas
* numpy
* lxml
* bokeh
